# PROJETO-ABPJ-3-Sistema-Gestor-Modular
Gestor Financeiro Pessoal.
Quais dados serão armazenados? rendimentos, gastos diarios, boletos a pagar e gastos diversos.
Quais funcionalidades o sistema terá? o sistema fará o calculo dos valore de receita subtraindo valores de gastos, boletos e gastos diarios.
lista de funcões que serão criadas: cadastrar,listar,calcular,remover e atualizar.
